Fedger App Installation Guide

Thank you for trying out Fedger, a personal finance app for tracking money you owe or are owed by friends, family, and acquaintances.

Important: This app is distributed outside of Google Play Store. You'll need to enable installation from unknown sources to install it.

How to Install Fedger on Android

  1. Transfer the fedger-1.0.apk file to your Android device
  2. On your Android device, go to Settings
  3. Navigate to Security or Privacy (may vary depending on your device)
  4. Find and enable the option "Unknown sources" or "Install unknown apps"
  5. If prompted to select an app, choose your file manager or browser
  6. Use your file manager to locate the fedger-1.0.apk file you transferred
  7. Tap the APK file to begin installation
  8. Follow the on-screen prompts to complete installation
  9. Once installed, you can open the app from your app drawer

Features of Fedger

  • Add and manage contacts you have financial dealings with
  • Record transactions (money given or received)
  • Track balances for each person
  • View transaction history
  • Search for contacts

Troubleshooting

If you encounter any installation issues:

  • Make sure you've enabled installation from unknown sources
  • Check if your Android version is 7.0 (Nougat) or higher
  • If you get a "Blocked by Play Protect" message, tap "Install Anyway"
  • If installation fails, try downloading the APK file again
